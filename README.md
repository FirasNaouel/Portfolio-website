# Portfolio Website

## Features

- Responsive, mobile-first layout
- Light and dark theme toggle
- Smooth scrolling navigation
- Animated skill bars and reveal effects
- Project showcase cards
- Contact form wired to Web3Forms

## Tech Stack

- HTML
- CSS
- JavaScript
- Typed.js (CDN)
- Font Awesome (CDN)
- Web3Forms (form handling)

## Project Structure

- index.html
- styles.css
- script.js
- assets/
- favicon_io/

## Getting Started

1. Open index.html in a browser, or
2. Serve the folder with a local server of your choice.

## Contact Form

The contact form posts to Web3Forms using the action URL and access key in index.html. To use your own account:

1. Create a Web3Forms account and generate an access key.
2. Replace the value of the hidden access_key input in index.html.

## Customize

- Update content in index.html (About, Projects, Contact).
- Update styles in styles.css.
- Update interactivity in script.js.
